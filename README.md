# city-code

中国大陆城市行政区划数据，包含**城市名称**、**行政区划代码**、**层级数据**、**拼音排序**、**父代码**等字段，数据来源: [中华人民共和国民政部](http://www.mca.gov.cn/article/sj/xzqh/1980/) .  
最新数据为 ``2020年12月01日 数据``

> 不包含港澳台数据

## 数据示例


| 名称  | 描述         | 示例          |
| ----- | ------------ | ------------- |
| level | 等级         | 省0市1区2.... |
| name  | 名称         | 北京市        |
| code  | 行政区划代码 | 110000        |
| initials  | 拼音首字母缩写 | bjs       |
| pinyin  | 完整拼音 | běijīngshì       |
| parent  | 父级代码 | 110000       |
